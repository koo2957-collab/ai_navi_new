## 1. AI 모델 설정

AI 기능을 사용하려면 API 키를 먼저 등록해야 합니다.  
**Gemini(구글)**를 기준으로 설명합니다. — 무료로 사용할 수 있어 처음 시작하기에 적합합니다.

### API 키 발급 방법 (구글 AI 스튜디오)

1. 크롬 브라우저에서 [https://aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey) 접속
2. 구글 계정으로 로그인
3. **"Create API Key"** 버튼 클릭
4. 생성된 키(`AIza...`로 시작)를 복사

### 프로그램에 등록하는 방법

1. 화면 오른쪽 상단 **`⋮` (더보기 메뉴)** 클릭
2. **"API 설정"** 선택
3. AI 서비스: **Gemini** 선택 (기본값)
4. API Key 입력란에 복사한 키 붙여넣기
5. **"연결 테스트"** 버튼 클릭 → "연결 성공" 메시지 확인
6. **"저장"** 버튼 클릭

> **참고:** API 키는 내 브라우저에만 저장됩니다. 백업·내보내기 파일에는 포함되지 않습니다.

## 2. 새 업무 만들기

### 방법 ①: 체크리스트 직접 생성

1. 왼쪽 사이드바의 **"새 업무"** 버튼 클릭
2. **"체크리스트 생성"** 선택

**입력 화면 사용법:**

| 항목 | 설명 |
|------|------|
| **제목** | 업무명 입력 (필수) |
| **업무내용** | 절차·목적·주의사항 등 간단히 작성 |
| **체크리스트 항목** | 처리해야 할 단계를 하나씩 입력 |
| **마감일** | 날짜 선택 (선택사항) |
| **폴더** | 저장할 폴더 선택 (선택사항) |

**체크리스트 항목 입력 단축키:**

| 키 | 동작 |
|----|------|
| `Enter` | 다음 항목 추가 |
| `Tab` | 현재 항목을 하위 항목으로 이동 |
| `Shift + Tab` | 현재 항목을 상위 항목으로 이동 |
| `⠿ 드래그` | 항목 순서 변경 |
| `Ctrl + Enter` | 저장 |

**AI로 체크리스트 자동 생성:**

1. "AI로 체크리스트 채우기" 영역 클릭하여 펼치기
2. 관련 문서(법령·지침·공문, PDF/TXT) 첨부 (선택)
3. **"AI로 채우기"** 버튼 클릭 → AI가 체크리스트 초안 자동 생성
4. 생성된 항목 확인 후 수정
5. **"저장하기"** 클릭

---

### 방법 ②: 업무 가져오기

다른 사람에게 공유받거나 이전에 내보냈던 업무 파일(`.json`)을 불러옵니다.

1. **"새 업무"** 버튼 클릭
2. **"업무 가져오기"** 선택
3. `.json` 파일 선택 → 자동으로 업무 목록에 추가

## 3. 업무 관리 · 폴더

업무가 많아지면 폴더로 분류하여 관리할 수 있습니다.

### 메인 화면 탭 필터

업무 목록 상단 탭으로 표시 범위를 전환합니다.

| 탭 | 표시 내용 |
|----|----------|
| **진행중** | 폴더 미배정 업무 (처리완료 제외) |
| **레퍼런스** | 폴더에 배정된 업무 |
| **처리완료** | 완료 처리된 업무 |
| **전체** | 모든 업무 |

### 업무 관리 화면 열기

1. 오른쪽 상단 **`⋮` 메뉴** 클릭
2. **"업무 관리"** 선택

### 새 폴더 만들기

1. 업무 관리 화면 오른쪽 상단 **"새 폴더"** 버튼 클릭
2. **"새 폴더 만들기"** 창이 열리면 생성 위치(상위 폴더) 선택
3. 폴더 이름 입력
4. **"새 폴더 만들기"** 버튼 클릭

### 업무 폴더로 이동

- 업무 목록에서 업무를 **우클릭** → "폴더로 이동" 선택
- 이동할 폴더를 클릭하면 완료

### 업무 관리 화면에서 할 수 있는 것

- 업무명 검색
- 상태 필터 (진행중 / 완료 / 지연 / 기한 임박)
- 정렬 (최신순 / 제목순 / 진행률순 / 기한순)
- 왼쪽 사이드바에서 항목 클릭 → 해당 항목 업무만 우측 테이블에 표시

**왼쪽 사이드바 항목:**

| 항목 | 설명 |
|------|------|
| **전체 업무** | 모든 업무 표시 |
| **폴더** | 해당 폴더 업무만 표시 |
| **미분류** | 폴더에 배정되지 않은 업무 |
| **아카이브** | 보관 처리된 업무 |
| **처리완료** | 완료 처리된 업무 목록 표시 |

### 업무 완료 처리 방법

모든 체크리스트가 완료된 업무를 "처리완료" 상태로 이동할 수 있습니다.

- 포커스 모드 상단 **"✅ 완료 처리"** 버튼 클릭
- 처리완료된 업무는 메인 화면에서 분리되어 **처리완료 탭**에서 확인

### 완료 취소 방법

처리완료된 업무를 다시 진행중으로 되돌릴 수 있습니다.

1. 업무 관리 화면 → 왼쪽 **"처리완료"** 클릭
2. 되돌릴 업무의 **`⋯`** 버튼 클릭
3. **"완료 취소"** 선택 → 진행중으로 복귀

## 4. 포커스 모드 — 상단 버튼 기능

포커스 모드는 업무 하나에 집중해서 처리하는 화면입니다.

### 포커스 모드 진입 방법

- 업무 카드 클릭 → 상세 팝업에서 **"체크리스트 편집"** 버튼 클릭
- 또는 업무 카드 오른쪽 **[포커스]** 버튼 클릭

### 상단 버튼 설명

| 버튼 | 기능 |
|------|------|
| **체크리스트 편집** | 체크리스트 항목을 추가·수정·삭제·순서변경 |
| **내보내기** | 이 업무를 `.json` 파일로 저장 (공유·백업용) |
| **AI 공문 기안** | AI가 이 업무의 공문서 초안을 자동으로 작성 |
| **체크리스트 출력** | 인쇄하거나 PDF로 저장할 수 있는 출력 화면 열기 |
| **✅ 완료 처리** | 업무를 처리완료 상태로 이동 (모든 항목 완료 후 활성화) |
| **삭제** | 업무 영구 삭제 (삭제 후 복구 불가) |

## 5. 포커스 모드 — 체크리스트 항목 세부 기능

포커스 모드에서 각 체크리스트 항목을 클릭하면 세부 기능을 사용할 수 있습니다.

### 탭별 기능 설명

| 탭 | 설명 |
|----|------|
| **업무내용** | 이 항목의 처리 방법, 절차, 주의사항 등을 자유롭게 기록 |
| **첨부자료** | 이 항목과 관련된 파일(공문·사진·문서)을 첨부 |
| **서식** | 이 항목 처리에 필요한 서식 파일을 첨부 (한글·Word·Excel 등) |
| **업무팁** | 나만의 처리 노하우, 참고사항을 메모 — AI가 자동으로 팁을 제안할 수도 있음 |
| **조치결과** | 처리 완료 후 결과 내용을 기록 — 출력 시 함께 인쇄됨 |

### 건너뛰기 기능

처리할 필요가 없는 항목을 '완료'로 표시하지 않고 건너뛸 때 사용합니다.

1. 항목 오른쪽 **`⋮`** 버튼 클릭
2. **"건너뛰기"** 선택
3. 건너뛰는 사유 입력
4. 확인 → 해당 항목은 진행률에 포함되지만 체크 없이 구분 표시됨

## 6. AI 기능 3종

### ① AI 길잡이 — 현재 업무 단계 가이드

체크리스트 항목 하나하나에 대해 AI가 처리 방법을 안내합니다.

**사용 방법:**
1. 포커스 모드에서 체크리스트 항목에 마우스를 올리면 오른쪽에 **"AI 길잡이"** 버튼 표시
2. 버튼 클릭 → AI가 해당 항목의 처리 방법, 관련 법령, 주의사항 등을 안내

---

### ② AI 길잡이 — 다음 업무 추천

체크리스트를 완료하면 AI가 다음에 처리할 업무를 자동으로 추천합니다.

**사용 방법:**
- 체크리스트 항목에 체크 표시할 때 자동으로 동작
- 화면 하단에 **"다음 업무 제안"** 팝업이 나타나면 내용 확인
- 추천 업무를 바로 등록하거나 닫기

---

### ③ AI 업무파트너 — 챗봇

행정 업무에 대해 자유롭게 AI에게 질문할 수 있는 채팅 기능입니다.

**사용 방법:**
1. 화면 오른쪽 하단 **AI 버튼** 클릭 → AI 패널 열기
2. 입력창에 질문 입력 후 `Enter` 또는 **"전송"** 버튼 클릭

**현재 업무 연결 기능:**
- 업무 상세 화면에서 AI 패널을 열면 **"현재 업무"** 토글이 자동으로 활성화됨
- 이 상태에서 질문하면 AI가 현재 업무 내용을 참고하여 답변

**모델 선택 (입력창 하단 좌측):**

| 모델 | 특징 |
|------|------|
| **자동** | 질문 내용에 따라 AI가 최적 모델 선택 (기본 추천) |
| Gemini 3.1 Flash Lite | 빠르고 가벼움 · 하루 500회 무료 |
| Gemini 3.5 Flash | 사고형(Thinking) 모델 |
| Gemini 2.5 Flash | 사고형(Thinking) 모델 |
| Gemini 3 Flash | 사고형(Thinking) 모델 · 최신 |
| Gemini 2.5 Flash Lite | 가벼운 사고형 · 하루 20회 무료 |
| Gemma 4 31B / 26B | 구글 오픈소스 · 하루 1500회 무료 |
| Gemini 3.1 Pro | 최고 성능 · 유료 |

## 7. 공유

업무를 다른 사람에게 전달할 때 사용합니다.  
**조치결과는 자동으로 제외**되고 체크 상태가 초기화된 파일이 생성됩니다.

### 공유 방법

1. 오른쪽 상단 **"공유"** 버튼 클릭
2. 공유할 업무 선택
   - 폴더 옆 체크박스: 해당 폴더 전체 선택
   - 전체 선택 / 전체 해제 버튼 활용
3. **"공유"** 버튼 클릭 → `.json` 패키지 파일 다운로드

### 받는 사람이 불러오는 방법

1. **"새 업무"** → **"업무 가져오기"**
2. 전달받은 `.json` 파일 선택 → 완료

> **참고:** 서식 파일(첨부된 한글·Word 등)도 패키지에 포함됩니다.

## 8. 설정 — 백업 · 용량 관리

데이터 손실을 방지하고 저장 공간을 관리합니다.

### 백업 · 용량 관리 화면 열기

1. 오른쪽 상단 **`⋮` 메뉴** 클릭
2. **"백업 · 용량 관리"** 선택

### 백업 (전체 내보내기)

- **"전체 내보내기"** 버튼 클릭
- 모든 업무·템플릿·폴더 정보가 `.json` 파일로 저장됨
- 이 파일은 복구용으로 안전한 곳에 보관

> **자동 백업:** 프로그램이 5분마다 자동으로 내부 백업을 생성합니다.  
> 화면 상단에 "자동 백업 완료" 표시가 뜨면 정상 동작 중입니다.  
> ⚠️ **주의:** 자동 백업은 업무 텍스트 데이터만 저장되며, **첨부파일·서식 파일·PDF는 포함되지 않습니다.**  
> 파일까지 완전히 보존하려면 **"전체 내보내기"** 로 `.json` 파일을 직접 저장하세요.  
> (전체 내보내기는 첨부파일·서식 파일·PDF 문서를 모두 포함합니다.)

### 백업 복구 방법

1. **"새 업무"** → **"업무 가져오기"**
2. 백업 파일 선택 → 데이터 복원

### 용량 확인

- 백업·용량 관리 화면에서 현재 사용 중인 저장 용량 확인 가능
- 브라우저 저장소(IndexedDB)를 사용하며, 일반적으로 1GB 이상 사용 가능
- 용량이 부족하면 처리완료된 오래된 업무를 삭제하여 공간 확보
